## 🚀 Como subir no GitHub Pages (de graça, em 5 minutos)

### 1. Crie uma conta no GitHub
Acesse [github.com](https://github.com) e crie sua conta se ainda não tiver.

### 2. Crie um repositório novo
- Clique em **"New repository"**
- Nome sugerido: `midia-kit`
- Deixe como **Public**
- Clique em **"Create repository"**

### 3. Faça upload do arquivo
- Na página do repositório, clique em **"uploading an existing file"**
- Arraste o arquivo `midia-kit-guhspm-v5.html`
- **Renomeie para `index.html`** antes de confirmar (importante!)
- Clique em **"Commit changes"**

### 4. Ative o GitHub Pages
- Vá em **Settings** (aba no topo do repositório)
- No menu lateral, clique em **Pages**
- Em "Branch", selecione **main** e clique em **Save**

### 5. Acesse seu site
Após 1–2 minutos, seu mídia kit estará disponível em:
```
https://SEU-USUARIO.github.io/midia-kit
```
Esse link pode ser enviado diretamente para marcas e parceiros.

---

## ✏️ Como atualizar o site

Sempre que quiser atualizar (novo vídeo, nova marca, métricas novas):

1. Gere o arquivo HTML atualizado
2. No repositório do GitHub, clique no arquivo `index.html`
3. Clique no ícone de lápis ✏️ ou em **"..."** → **"Upload new file"**
4. Faça upload do novo arquivo (já com o nome `index.html`)
5. O site atualiza automaticamente em instantes
